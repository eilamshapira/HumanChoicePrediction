# Human Choice Prediction in Non-Cooperative Games: Simulation-based Off-Policy Evaluation

Running instructions:
1. Install the Anaconda environment using the file "init_env.sh".
2. Create a wandb sweep using the file "wandb.yaml".
3. Paste the name of the sweep in the appropriate place in the file "run.sh", and run it.

To create a simulation dataset and save it to a csv file, use the "create_csv_of_simulation.ipynb" notebook.
